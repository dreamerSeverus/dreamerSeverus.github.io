系列目錄

前記

嗯，怒氣再次被蓄滿，可以收拾收拾準備剛一波正面了。準備密集的刷一波題，修煉一下技術。招聘季與否，扎實的基礎對於IT生涯的重要性強調10000遍也不為過。扎實的基礎包括：職業方向核心的1~2門語言，從底層核心庫到常用必備應用框架的實現原理、應用示例。數據結構與算法，包括常見面試題考法及變種。系統設計原理，包括常見後端方向技術框架、分佈式系統原理、特性、實現，要求具備基礎的原理闡釋、命題式實現與架構能力。特長系列，即個人偏好技術方向探索及實例工作，如相關技術方向研究、源碼分析及取得成果。沒有成果皆是口水流，BB俠。此文章算做工作計劃、亦可作為來日見證。

一、語言基礎系列

Java

本人大二開始接觸Java，已近五年，心性使然，並未沿Java路線持續推進，功底不深，平日開發以任務完成為核心，學習路線雜糅不專。然而後期對大數據處理相關分佈式系統興趣濃厚，如Hadoop、Yarn等，總體說來仍然是Java後端處理路線，重拾和專注Java亦是必然。此系列博客，以面試常見考點為綱要，包含Java基礎總結（多線程、NIO、並發、反射等）、以及三大框架SSH相應原理及考點。此外，JVM作為Java基礎，在後期開發和系統調優方面，幾乎是必備基礎，在此一併總結。Java作為一門面向對象語言，在系統後端一直應用廣泛，嗯，所以，就複習加學習啦。

python

python作為一門出色的腳本語言，同樣應用廣泛，在Web開發、系統後端構建、數據分析等領域均表現出色。聽說了”Life is too short, I use python”，所以就決定開動了。在研究生期間短暫參與的平安大數據比薩中，零散學習了python，算是入了門，實習期間，也是根據作業需要，短暫學習和寫了部分python代碼。因此，在此處對python做相應系統總結，建立知識樹。

Scala

Scala是接觸Spark后開始了解的一門語言，在跑Spark例程時，發現繁複的一頁MapReduce代碼，可以通過寥寥幾行Scala代碼搞定。其語言結合了面向對象特點與函數式編程特性，運行與Java虛擬機之上，作為Spark的開發語言，必將在後期蓬勃發展。因此，此處同樣系統總結Scala基礎知識，作為探索Spark底層的一項重要基礎。

二、數據結構與算法

數據結構與算法，是計算機領域的最重要基礎。本人一直在此處翻跟頭，因此將重點對此處進行學習及總結。數據結構與算法的複習、準備，應該包括基礎原理的明了、代碼實現，以及工業界類庫實現的探索、研究。在準備面試過程中，需要以在基礎明了的前提下，系統刷題，包括LeetCode、牛客網以及程序員面試金典或是江湖流傳的July博客、劍指offer等。在完成的前提下，可以考慮經典書籍，如編程珠璣等或是ACM必備的各類題庫：POJ等。本文僅對個人刷題路線進行總結，預計以牛客網和LeetCode為主。

三、系統設計原理

偶然的機會在微博上發掘到了九章算法設計班。在更高級的刷offer路線上，至少BAT或Facebook、Twitter、Google等互聯網外企面試過程中，系統設計也是必備的知識，包括分佈式系統設計題目、數據庫設計、並發系統策略、文件系統選擇等。暑假期間，斷斷續續聽完了相應的課程，攢下了講義，然而之後並未有時間繼續翻過，加深理解。此系列博客將以九章算法設計班涉及的內容為綱要，對涉及的知識和原理做深化與細化總結，也為相應職業生涯鋪平道路。

四、特長系列

由於種種原因，本人對Spark情有獨鐘。從2013年偶爾了解到Spark作者的相關履歷后，便對Spark技術保持持續關注。在大數據井噴的時代，Spark也當之無愧的成為了中流砥柱。其核心的RDD設計，使得核心計算及IO均在內存中進行，大大提高了機器學習算法在大規模分佈式系統中的應用範圍。本人在2015年9月至10月初層密集學習過一段Spark,包括環境搭建、數據分析及源碼閱讀，然而由於畢業論文等其它事務導致中止。從本系列博客開始，將重拾相應技術體系，作為一項愛好，持續保持下去。以Spark為載體，通過對Spark原理、機制的探索學習，打通分佈式系統設計底層，是本人的初衷與願景。

結束語

BB了許多，終究是些計劃和願景，並無卵用。然本人有心以此文為證，夯實基礎，于紛繁的技術世界中走出自己的一條路來。其中過程，讀者可鑒。雄關漫道真如鐵，而今邁步從頭越。以文字與代碼為證，且看且行。
